# 文档树压缩
## 功能
 - 自定义文档树中（子）文档项目的缩进
 - 自定义文档树中文档项目图标的隐藏与否
 - 自定义文档树的文档项目字体大小
 - 文档树项目流动布局：在鼠标悬停时，暂时将缩进取消，避免多级子目录难以看到标题或难以点击
